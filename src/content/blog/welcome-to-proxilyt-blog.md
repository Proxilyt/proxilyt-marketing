# Welcome to Proxilyt Blog!

This is your first blog post. You can add as many markdown (.md) files as you want to the `src/content/blog/` folder and they will automatically appear on your blog.

## How to Add Blog Posts

1. Create a new `.md` file in `src/content/blog/`
2. Add the required frontmatter (title, description, pubDate)
3. Write your content in markdown
4. The post will automatically be published!

## Example Frontmatter

```yaml
---
title: Your Post Title
description: A brief description
pubDate: 2026-02-19
author: Your Name
tags: [tag1, tag2]
---
```

Happy blogging!
